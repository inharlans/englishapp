## 발견/반영 체크리스트 (10건)
1. 퀴즈 문제 로드 후 입력창 포커스가 자동 이동되지 않아 직접 풀이 속도가 느림 -> 정답 입력창 자동 포커스 추가
2. 오답 후 `다시 풀기` 시 포커스가 복귀되지 않아 키보드 풀이 흐름이 끊김 -> 재시도 시 입력 포커스 복귀
3. 제출 버튼이 빈 입력에서도 활성화되어 불필요한 요청 가능 -> `trim` 기준 입력 없으면 비활성화
4. 퀴즈 풀이 중 문제를 잠시 넘기기 어려움 -> `건너뛰기` 버튼 추가
5. 세션 진행 품질(정답률) 확인이 어려움 -> 헤더에 정답률(정답/시도) 표시
6. 파트 내 진행 정도 시각화가 부족함 -> 파트 진행률 바 추가
7. 파트 변경 시 이전 피드백/메시지가 남아 혼동 유발 -> 파트 변경 시 피드백/메시지 초기화
8. 퀴즈 미로그인 상태에서 즉시 로그인 동선 부족 -> 로그인 링크 추가
9. 다운로드 미완료 접근 차단 시 상세 페이지 복귀 동선 부족 -> 단어장 상세 이동 링크 추가
10. 실사용 점검 기준으로 `/wordbooks/[id]` 퀴즈 루프(문제 풀이→피드백→다음)의 키보드 중심 흐름을 우선 보강

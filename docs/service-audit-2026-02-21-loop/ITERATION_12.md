## 발견/반영 체크리스트 (10건)
1. 오프라인 라이브러리 새로고침/삭제 성공 피드백이 없어 상태 추적이 어려움 -> 정보 배너(`info`, `role=status`) 추가
2. 삭제 확인 문구가 대상 단어장 정보를 담지 않아 오삭제 위험이 있음 -> 제목 포함 확인 문구로 변경
3. 검색/정렬 활성 상태가 결과 요약에 충분히 드러나지 않음 -> 정렬/검색어/필터 적용 여부를 결과 문구에 표시
4. 검색 입력/정렬 셀렉트의 목적 라벨이 약함 -> `aria-label` 보강
5. 오프라인 목록 빈 상태에서 마켓 이동 동선 부족 -> `마켓 둘러보기` 링크 추가
6. 카드에서 온라인 원본 진입 동선이 없음 -> `온라인 원본` 버튼 추가
7. 오프라인 학습 페이지에 키보드 내비게이션 부재 -> ←/→/Space 단축키 지원
8. 오프라인 학습 진행률 가시성이 낮음 -> 진행률 바(% 표시) 추가
9. 오프라인 학습 로딩/에러/정보 메시지 접근성 약함 -> `role=status`/`role=alert` 정리
10. 단어 0개 오프라인 단어장 처리 메시지 부재 -> 빈 카드 상태 안내 + 온라인 원본 확인 링크 추가

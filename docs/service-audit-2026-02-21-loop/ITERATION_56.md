# ITERATION 56 (2026-02-22)

## Scope
- MCP 브라우저 점검: `https://www.oingapp.com/wordbooks/new`
- `app/wordbooks/new/page.tsx` 깨진 문자열/문구 복구

## MCP 확인 결과
- 배포본에서 생성 페이지 문구는 정상 노출되어야 하나, 소스에는 인코딩 깨짐 텍스트가 다수 존재했음
- 가이드/상태 메시지/버튼 라벨이 손상되어 유지보수 위험이 높았음

## 적용한 개선점 (10+)
1. `wordbooks/new` 페이지를 동일 기능으로 재작성해 깨진 문자열 제거
2. TSV/CSV 예시 데이터를 정상 한국어 문구로 복구
3. `term 필수`, `meaning 필수` 검증 메시지 정상화
4. 붙여넣기 파싱 완료 메시지 정상화
5. 붙여넣기 초기화 메시지 정상화
6. 업로드 성공/실패 메시지 정상화
7. 클립보드 복사 실패 메시지 정상화
8. CSV 템플릿 다운로드 완료 메시지 정상화
9. 생성/저장 실패 플래시 메시지 정상화
10. 섹션 타이틀/입력 라벨/탭 라벨 전부 정상화
11. 수동 입력/검증 테이블의 aria-label 및 동작 라벨 정상화
12. 하단 상태 영역을 상시 렌더링해 레이아웃 점프 완화

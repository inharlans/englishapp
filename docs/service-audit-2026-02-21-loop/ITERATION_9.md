## 발견/반영 체크리스트 (10건)
1. 네비게이션 로그인 링크가 현재 경로 복귀를 보장하지 않음 -> `next` 쿼리로 현재 경로 전달
2. 계정 버튼 그룹 의미가 약함 -> 계정 영역을 `role=group` + 라벨로 명확화
3. 로그아웃 링크 목적 전달이 약함 -> 로그아웃 링크 `aria-label` 보강
4. 홈의 1단계/2단계 카드가 목록 구조로 노출되지 않음 -> `ul/li` 구조로 변경
5. 홈 CTA 묶음의 컨텍스트가 불명확함 -> 게스트/로그인 사용자별 `role=group` 라벨 부여
6. 홈 정책 링크 블록이 내비게이션 의미를 갖지 않음 -> `nav` 랜드마크 적용
7. 로그인 사용자 빠른 이동 카드가 일반 `div`로만 구성됨 -> `nav` 랜드마크 적용
8. 오프라인 검색 입력 보조 설명이 없어 사용성이 떨어짐 -> `aria-describedby` + 도움 문구 추가
9. 오프라인 빈 상태에서 복구 동선이 약함 -> 항목 없음 시 내 단어장 링크, 검색 결과 없음 시 검색어 초기화 액션 추가
10. 오프라인 목록/버튼 의미 전달이 약함 -> 목록 `role=list/listitem` + 삭제/학습 버튼 `aria-label` 보강

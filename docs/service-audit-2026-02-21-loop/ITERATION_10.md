# 10회차 점검 보고서 (2026-02-21)

## 점검 범위
- `/wordbooks/market` 검색/로그인/페이지네이션 동선
- `/wordbooks/[id]` 메타 정보/목록/페이지네이션 접근성
- 결제 로직 및 결제 API 제외

## 발견/반영 체크리스트 (10건)
1. 마켓 상단 로그인 버튼이 현재 필터/페이지 컨텍스트를 잃음 -> 현재 쿼리를 `next`에 포함해 로그인 복귀 보장
2. 마켓 빈 상태 로그인 CTA가 컨텍스트 복귀를 보장하지 않음 -> 현재 검색 쿼리 기반 로그인 URL로 교체
3. 마켓 카드의 게스트 다운로드 CTA가 상세 페이지로만 보내 일관성이 약함 -> 마켓 컨텍스트 복귀 로그인 링크로 통일
4. 마켓 필터 UI에 즉시 초기화 동작이 없어 재탐색 비용이 큼 -> 필터 폼 내 `필터 초기화` 링크 추가
5. 마켓 모바일 페이지네이션의 이전/다음 `aria-label`이 일반 문구여서 목적성이 약함 -> 실제 이동 페이지 번호 기반 라벨로 보강
6. 단어장 상세의 다운로드일/동기화일이 UTC 슬라이스 표기로 노출됨 -> `Asia/Seoul` 기준 날짜 포맷으로 통일
7. 단어장 상세 상단 액션 그룹 의미가 불명확함 -> `role=group` + 라벨 추가
8. 단어 목록 0건일 때 소유자에게 다음 행동 안내가 부족함 -> `단어 추가` 영역 유도 문구 추가
9. 단어 목록 컨테이너가 목록 구조를 갖지 않음 -> `role=list/listitem` 보강
10. 단어 상세 페이지의 이전/다음 페이저가 비활성 링크(`pointer-events-none`) 방식이라 접근성 저하 -> 비활성 `span` + 활성 링크 분기 렌더링으로 개선

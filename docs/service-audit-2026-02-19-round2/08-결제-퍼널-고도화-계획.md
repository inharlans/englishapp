# 결제 퍼널 고도화 계획

## 목표

- 가격 열람 -> 로그인 -> 결제 완료 전환율을 높인다.

## 실행 항목

1. `/pricing`에 혜택 비교 FAQ/환불/해지 안내 정교화
2. 비로그인 CTA 클릭 시 `next=/pricing` 보존 검증 강화
3. 결제 실패 사유를 사용자 친화 문구로 분류 노출
4. 성공/취소 이후 복귀 동선(추천 액션) 추가

## 지표

- 가격 페이지 진입 대비 로그인 전환율
- 로그인 후 결제 시도율
- 결제 성공률/취소율

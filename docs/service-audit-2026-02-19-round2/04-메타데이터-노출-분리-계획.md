# 메타데이터 노출 분리 계획

## 목표

- 내부 기술 메타(`Imported from ...`)를 사용자 화면에서 제거한다.

## 문제

- 다운로드 목록/상세에 개발성 출처 문자열이 그대로 노출됨

## 실행 항목

1. `displayDescription`(사용자용)와 `internalSource`(운영용) 분리
2. 사용자 화면은 `displayDescription`만 렌더링
3. 관리자 페이지에서만 원본 source 확인 가능하게 분리
4. 기존 데이터 마이그레이션 스크립트 실행

## 완료 기준

- 사용자 페이지에 기술 경로/파일명 노출 0건

## 2026-02-19 처리 결과
- 반영 완료: 사용자용 displayDescription, 내부 source 분리 렌더링 적용

# 관리자 검수 워크플로우 확장 계획

## 목표

- 마켓 품질 관리 작업을 수동 대응에서 운영 프로세스로 전환한다.

## 실행 항목

1. 신고 상태 확장: `OPEN/REVIEWING/RESOLVED/DISMISSED`
2. 조치 템플릿(숨김/수정요청/게시중지) 표준화
3. 단어장 품질 점수(텍스트 품질, 신고 이력, 다운로드 만족도) 도입
4. 승인 큐 기반 게시 프로세스 운영

## 완료 기준

- 신고 평균 처리시간 24시간 이내
- 반복 신고 콘텐츠 재발률 50% 이상 감소

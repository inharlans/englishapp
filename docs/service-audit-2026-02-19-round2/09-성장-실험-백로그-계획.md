# 성장 실험 백로그 계획

## 목표

- 학습 지속률/다운로드 전환률을 빠르게 개선한다.

## 실험 백로그

1. 마켓 카드에 “학습 난이도/추천 대상” 뱃지 추가
2. 다운로드 직후 3일 온보딩 시퀀스(암기->퀴즈->회복)
3. 학습 종료 후 다음 행동 추천 카드 A/B 테스트
4. 오프라인 저장 유도 위치/문구 실험
5. 리뷰 작성 타이밍 실험(첫 20문항 완료 시점)

## 측정 지표

- D1/D7 학습 재방문

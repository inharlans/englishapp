# 1. 카드 밀도를 사용자가 직접 조절하는 ‘컴팩트/표준/집중’ 보기 모드를 제공한다.

## 2. 문제점 설명
모든 사용자가 같은 카드 크기와 정보 밀도를 선호하지는 않습니다. 초보자는 넓고 친절한 화면이 필요하고, 숙련자는 더 많은 단어를 빠르게 보고 싶어합니다. 단일 레이아웃은 둘 중 한쪽 경험을 희생하게 됩니다.

## 3. 제안된 해결 방안
- 화면 우측 상단에 보기 모드 토글(`컴팩트/표준/집중`) 배치
- 모드별로 카드 패딩, 글자 크기, 예문 표시 여부를 다르게 적용
- 선택한 모드를 사용자 로컬 설정으로 저장해 재방문 시 유지
- 모바일에서는 모드 전환 시 레이아웃 점프를 줄이는 부드러운 전환 애니메이션 적용

## 4. 예상 효과
사용자 성향과 학습 상황에 맞는 정보 밀도를 제공해 체감 사용성이 크게 좋아진다. 초보자는 부담 없이 읽고, 숙련자는 더 빠르게 반복 학습할 수 있어 만족도가 동시 개선된다. 개인화된 화면 제어권은 제품 완성도와 애착을 높이는 핵심 요소로 작동한다.

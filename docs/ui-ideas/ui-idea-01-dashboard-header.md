# 1. 정보 우선순위를 한눈에 보여주는 ‘학습 대시보드 헤더’로 첫 화면 집중도를 높인다.

## 2. 문제점 설명
현재 화면은 기능 버튼과 정보가 섞여 있어 사용자가 무엇을 먼저 봐야 하는지 판단하기 어렵습니다. 중요한 학습 상태와 다음 행동이 동일한 시각 무게로 배치되면 집중도가 떨어집니다. 결과적으로 첫 진입 시 탐색 시간이 늘어납니다.

## 3. 제안된 해결 방안
- 헤더 상단을 `오늘의 진행률`, `연속 학습일`, `다음 추천 행동` 3개 카드로 고정
- 액션 버튼은 우선순위 1개(Primary) + 보조 2개(Secondary)만 노출
- 카드 내 아이콘/색상 규칙을 통일해 상태 해석 속도 개선
- 모바일에서는 카드 1열 스와이프, 데스크탑은 3열 고정 레이아웃 적용

## 4. 예상 효과
첫 화면에서 사용자가 지금 상태와 다음 행동을 즉시 이해해 클릭 의사결정이 빨라진다. 정보 위계가 명확해지면 UI가 깔끔해 보이고 학습 시작까지의 체감 시간이 줄어든다. 신규 사용자도 복잡하게 느끼지 않아 이탈을 줄이고 재방문 시에도 빠르게 루틴에 진입할 수 있다.

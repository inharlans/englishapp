# 제안서 02: 마켓 신뢰도/탐색성 개선

## 배경
- 마켓 목록 다수가 `설명이 없습니다`, 평점 `0.0(0)` 상태.
- 유사한 시리즈 단어장이 길게 나열되어 초반 탐색 효율이 낮음.

## 문제
- “무엇을 먼저 받아야 하는지” 판단 근거가 부족함.
- 콘텐츠가 많아도 품질 신호가 약해 신뢰가 떨어짐.

## 제안
- 카드 상단에 핵심 배지 추가: `추천`, `신규`, `많이 다운로드`, `검수완료`.
- 시리즈 묶음 표시(예: NGSL 1~600/601~900를 하나의 시리즈로 접기/펼치기).
- 설명 미입력 단어장은 기본 템플릿 요약 자동 생성.
- 정렬 기본값을 “인기순” + “초보 추천” 섹션 분리.

## 구현 범위
- 대상: `app/wordbooks/market/page.tsx`, 마켓 API 정렬/집계 로직
- 시리즈 메타 테이블(또는 title 규칙 파서) 도입

## 성공 지표
- 마켓 진입 후 첫 다운로드 전환율 증가
- 마켓 이탈률 감소

# 3회차 점검 보고서 (2026-02-19)

## 실제 점검 동선
- 마켓(`/wordbooks/market`) 1~2페이지 탐색
- 검색/정렬 폼 반복 사용

## 제안 5건
1. 마켓 검색이 제목/설명만 대상으로 동작해 제작자 기준 탐색이 불가능하다.
2. 단어장 규모(단어 수) 기반 필터가 없어 입문/중급 사용자 동선이 길다.
3. 페이지 이동 시 필터 상태가 URL에 모두 반영되지 않으면 재현성이 떨어진다.
4. 필터 초기화 링크가 확장된 필터 상태를 명시적으로 리셋해야 한다.
5. 대량 목록(30+개)에서 단어 수 구간 필터는 필수 탐색 축이다.

## 이번 회차 반영
- [완료] `app/wordbooks/market/page.tsx`
  - 검색 대상에 제작자 이메일 추가
  - 단어 수 구간 필터(`전체/100~300/301~700/701+`) 추가
  - 페이지네이션 링크에 `size` 필터 상태 유지
  - 필터 초기화 URL에 `size=all` 명시

# 5회차 점검 보고서 (2026-02-19)

## 실제 점검 동선
- 홈(`/`) 재진입 후 학습 시작 경로 확인
- 로그인 사용자 관점의 바로가기 효율 점검

## 제안 5건
1. 로그인 사용자는 홈에서 바로가기가 부족해 상단 메뉴 의존도가 높다.
2. 학습 재개/마켓 탐색/오프라인 복습은 홈에서 병렬 액션으로 보여줘야 한다.
3. 재방문 사용자에게는 "지금 할 일" 중심 카피가 필요하다.
4. 홈에서 즉시 이동 경로가 보강되면 첫 액션까지 시간이 단축된다.
5. 로그인 사용자 CTA가 단일 버튼이면 기능 발견성이 낮아진다.

## 이번 회차 반영
- [완료] `app/page.tsx`
  - 로그인 사용자 대상 빠른 이동 버튼(내 단어장/마켓/오프라인) 추가
  - 우측 패널에 3분기 바로가기 카드 추가
  - 재방문 사용자 설명 문구를 실행 중심으로 개선

## 종료 메모

# 1. 학습 종료 후 다음 행동을 추천하는 회고 패널로 재방문 루프를 만드는 흐름

## 2. 문제점 설명
사용자가 한 세션을 마치면 다음에 무엇을 해야 할지 안내가 약해 학습이 단발성으로 끝나기 쉽습니다. 정답/오답/회복 결과는 제공되지만 행동 제안이 없으면 동기와 루틴이 끊깁니다. 특히 신규 사용자는 성과 해석이 어렵습니다.

## 3. 제안된 해결 방안
- 세션 종료 시 `오늘의 회고 패널` 자동 노출
- 결과 기반으로 `오답 복습`, `회복 목록`, `다음 퀴즈` 중 1순위 추천 제공
- 예상 소요 시간(예: 3분, 7분)과 기대 효과를 함께 표시
- `내일 같은 시간 알림` 토글로 재진입 예약

## 4. 예상 효과
학습 종료 시점의 이탈을 다음 행동으로 전환해 세션당 학습량과 재방문 빈도를 동시에 높일 수 있다. 사용자는 단순 점수 대신 구체적 다음 단계와 소요 시간을 받아 실행 장벽이 낮아지고, 서비스는 루틴 형성에 유리한 경험을 제공해 장기 잔존율 개선 효과를 기대할 수 있다.

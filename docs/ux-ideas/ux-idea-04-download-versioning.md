# 1. 다운로드한 단어장의 최신 상태 동기화 여부를 명확히 보여주는 버전 UX

## 2. 문제점 설명
사용자는 다운로드한 단어장이 최신인지, 작성자가 업데이트했는지 알기 어렵습니다. 읽기 전용 정책은 명확하지만 버전 정보가 없으면 신뢰와 통제감이 떨어집니다. 오래된 데이터로 학습해도 사용자는 이를 인지하지 못할 수 있습니다.

## 3. 제안된 해결 방안
- 단어장 헤더에 `내 버전`과 `최신 버전` 표시
- 최신 버전이 있으면 `변경점 보기`와 `업데이트 적용` CTA 제공
- 업데이트 시 사용자 학습 상태 보존 여부를 선택(유지/초기화)
- 업데이트 이력(추가/수정/삭제 개수) 요약 제공

## 4. 예상 효과
사용자는 다운로드 데이터의 신선도를 명확히 파악해 더 신뢰하고 학습할 수 있다. 업데이트 전 변경점을 확인하고 학습 상태 보존을 선택할 수 있어 통제감이 높아지며, 운영 측면에서도 버전 혼선을 줄여 문의를 감소시킨다. 결과적으로 다운로드 기능의 실사용 가치가 커진다.

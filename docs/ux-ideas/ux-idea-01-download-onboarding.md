# 1. 다운로드 후 즉시 학습 시작 동선을 한 화면에서 완결하는 온보딩 배너

## 2. 문제점 설명
사용자는 마켓에서 단어장을 다운로드한 뒤 어디로 이동해야 학습을 시작하는지 즉시 이해하지 못하는 경우가 있습니다. 다운로드 성공 피드백이 단순하면 행동 전환(암기/퀴즈/리스트)까지 이어지지 않아 이탈이 발생합니다. 특히 모바일에서는 경로 탐색 부담이 더 큽니다.

## 3. 제안된 해결 방안
- 다운로드 완료 토스트 대신 상단 고정 온보딩 배너를 1회 노출
- 배너에 `Memorize 시작`, `Quiz Meaning`, `Quiz Word`, `나중에` 4개 CTA 제공
- CTA 클릭 시 해당 단어장의 내부 학습 경로(`/wordbooks/[id]/...`)로 즉시 이동
- 사용자가 닫으면 같은 단어장에 대해 24시간 재노출 제한

## 4. 예상 효과
다운로드 직후의 사용자가 다음 행동을 고민하지 않고 즉시 학습 화면으로 전환할 수 있어 첫 세션 전환율이 상승한다. 탐색 비용이 줄어 신규 사용자도 기능 구조를 빠르게 이해하고, 다운로드만 하고 떠나는 비율을 낮춰 실제 학습 시작률과 재방문율을 함께 개선할 수 있다.

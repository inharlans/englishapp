# 1. 단어장 상세에서 학습 탭을 고정해 기능 탐색 부담을 줄이는 구조

## 2. 문제점 설명
현재 사용자는 단어장 상세에서 여러 액션 버튼을 보고 필요한 기능을 직접 해석해야 합니다. 버튼이 늘어나면 정보가 평면적으로 나열되어 우선순위를 파악하기 어렵고, 반복 사용 시에도 시선 이동이 커집니다. 기능 인지는 되지만 사용 속도가 느려집니다.

## 3. 제안된 해결 방안
- 상세 상단에 `Memorize / Quiz / List` 탭 네비게이션 고정
- `Quiz`와 `List`는 하위 세그먼트(Meaning/Word, Correct/Wrong/Half)로 2단 구성
- 현재 위치 탭을 컬러/언더라인으로 명확히 표시
- 마지막으로 사용한 탭을 단어장별로 로컬 저장해 재진입 시 복원

## 4. 예상 효과
기능이 버튼 묶음이 아니라 학습 단계로 인식되어 사용자가 목적지까지 더 짧은 클릭으로 이동한다. 반복 학습 시 탭 복원으로 루틴 진입 시간이 줄고, 화면 전환의 예측 가능성이 높아져 전체 UX가 안정적으로 느껴진다. 결과적으로 탐색 피로도와 오클릭이 줄어든다.

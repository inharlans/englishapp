# 1. 품사·의미·예문을 카드 내 정보 우선순위로 재배치하는 가독성 개선

## 2. 문제점 설명
뜻 표기에서 `(명)(동)(형)` 같은 정보가 텍스트 흐름 안에 섞여 있으면 빠른 인지가 어렵습니다. 학습 중에는 짧은 시간에 핵심 정보를 스캔해야 하는데, 현재 구조는 의미와 보조정보의 시각적 구분이 약합니다. 결과적으로 카드 당 처리 시간이 늘어납니다.

## 3. 제안된 해결 방안
- 품사는 배지 스타일(예: `명사`, `동사`)로 의미 텍스트와 분리
- 의미 본문은 줄 간격 확대, 예문은 별도 블록 배경으로 레벨 분리
- 모바일에서 품사 배지는 1줄 스크롤 칩 형태로 처리
- 사용자 설정에 `간결 보기/자세히 보기` 토글 제공

## 4. 예상 효과
카드에서 무엇이 핵심 의미인지 즉시 구분되어 인지 부하가 줄고, 학습 속도가 개선된다. 특히 모바일에서 좁은 폭에서도 의미·품사·예문이 혼합되지 않아 오독 가능성이 감소하며, 개인화된 보기 밀도 설정으로 초보자와 고빈도 사용자 모두의 만족도를 높일 수 있다.

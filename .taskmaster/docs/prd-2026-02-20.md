# Englishapp PRD (2026-02-20)

## 목표
- 한국어 사용자 대상 영어 학습 웹앱에서 학습 완주율과 결제 전환율을 개선한다.

## 범위
1. 결제(PortOne) 마무리
- `/pricing`에서 월간/연간 결제 정상 동작
- 결제 성공/취소 리다이렉트 및 사용자 플랜 반영 검증
- 필수 환경변수: `PORTONE_CHANNEL_KEY`

2. 퀴즈 품질 개선(P1)
- 오답 재출제 큐 자동화
- 채점 진단 이벤트 수집
- 오답 상세 근거(입력값 vs 허용 후보 유사도) 노출

3. 운영 가시성
- 관리자 지표에서 퀴즈 오답 재검토 후보 비율 확인 가능

## 기능 요구사항
- 의미 퀴즈 오답 시 허용 답안 예와 채점 근거를 제공한다.
- 세션 중 오답 문항은 큐에 쌓여 재출제된다.
- 서버는 오답 건에 대해 진단 이벤트를 남긴다.
- 관리자 지표 API는 24시간 기준 재검토 후보 오답 비율을 계산한다.

## 비기능 요구사항
- 기존 타입체크/테스트/빌드를 통과한다.
- 모든 텍스트 파일은 UTF-8 인코딩을 유지한다.

## 수용 기준
- `npm run typecheck`, `npm run test -- lib/text.test.ts`, `npm run build` 통과
- `/api/admin/metrics` 응답에 `quizQuality` 필드 존재
- 퀴즈 오답 시 UI에 채점 근거 표시
